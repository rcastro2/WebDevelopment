Errors in programs

This example demonstrates the three types of errors you can have in a program: Syntax, Runtime and Logical errors.

Scenario
Calculate and display the average of three grades that the user enters.

Syntax Errors

These types are physical mistakes in the way instructions were written such as spelling mistakes, missing or incomplete parentheses, quotations or brackets. These errors stop a program from running

Runtime Errors

As the name implies these errors occur when the program is running but a situation arises that stop the execution of the program such as invalid user input or incorrect calculation that results in an illegal operation. These errors allow the program to run under normal conditions but stop when an illegal operation is encountered.

Logical Errors

These errors are typically the toughest to fix because they do not stop your program nor produce error messages to the console but still produce the wrong outcome These errors will not stop a program.
