Charts

This example demonstrates how to generate charts using the c3js JavaScript library. c3js library is built using the d3js JavaScript library, hence both libraries must be loaded. There are many types of charts with various customizations that can be applied to each chart. We will focus on Line, Bar and Pie charts with basic features.
